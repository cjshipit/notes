# node.js
 node.js 相当于一个javascript的解释器 可以不依赖浏览器直接运行javascript代码
 
### npm包管理
#### 介绍
   node.js自带的npm包管理和python的类似，很方便的下载,导入以及发布包
#### 使用
```
1 下载node.js
2 将npm的镜像改为淘宝:
  npm config set registry https://registry.npm.taobao.org (修改镜像)
  npm config get registry (查看镜像)
     
```
